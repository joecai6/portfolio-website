KIWI Forum
×

KIWI Forum is a web application that introduces a secure forum for employees to organize themselves into specialized teams and ask questions or discuss topics with experts at the subject. It features a specialization tagging function that allows employees to organize into groups and publicly display all specializations within the company. The tagging system also extends to posts where posts can be effectively filtered and searched through specializations. The main goal of the forum is to provide a platform for employees to communicate and share their knowledge.

HathiTrust Info & Search Tool
×

HathiTrust search tool provides quick access to research and historical records in the HathiTrust Digital Library. Along with keyword search for a specific record, an excel sheet can be processed to search for multiple records at the same time. There are also graph visualizations of the data such as the number of records published in a specific year and the number of records published by state. The search and graphs use all of the six million records available publicly for download in the HathiTrust dataset.

Algorithm Visualization
×

Graph algorithms such as BFS and DFS are visualized through a randomly generated 2D maze. The graphical interface allows the user to customize the size of the maze map and the algorithm used to find the end point. The goal of the project is used to visualize how certain algorithms work in a maze. Certain pathfinding algorithm like A* search and greedy best first search solve the maze faster than algorithms like BFS.

ACM AI Hide and Seek Bot
×

The Hide and Seek bot is a submission for UCSD ACM’s first AI competition which was centered around Hide and Seek. The submission consists of two bots, one seeker bot and one hider bot. The seeker bot uses pathfinding algorithms to chase the hider optimally and graph algorithms to explore the map in search for the hider. The hider bot wins by finding the closest island of walls and circles around it which blocks the vision of the seeker and runs away at the same time. The bot submission won first place in the competition and was recorded as part of ACM’s AI competitions.

Designed with Android Studio, Grade Calculator is a simple and appealing mobile application used to calculate GPA after a school semester and overall GPA. File Compressor
×

File Compressor compresses and decompresses text, image, and video files efficiently through the implementation of the Huffman encoding algorithm. The command line interface allows users to input a file to be compressed, then the program generates a compressed file with a new file extension.
